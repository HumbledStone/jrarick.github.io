Employee Directory

I created this project using AJAX to generate a directory of sample employee information. The app fetches randomly generated JSON data and displays employee avatars, e-mail addresses, and location. If the employee's box is clicked on, a pop up window will appear with additional info about the employee.

SVG Animation

This project was created to showcase different ways Scalable Vector Graphics can be used. SVGs are used as navigation icons, a repeating background image, a main logo with a hover effect, and a repeated SVG coded as a symbol that can have altered properties for each graphic.

Responsive Design Model Website

This project uses a mobile-first approach designed to ensure an optimized experienced at any screen size. I utilized media query breakpoints for mobile, tablet, and desktop screen sizes to guarantee content would accessible and look good on any device.
